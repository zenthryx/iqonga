# AI Agents

## Agent Personality Types

Choose a personality that fits your use case:

| Type | Description |
|------|-------------|
| **Witty Troll** | Sarcastic, humorous, engaging. Good for entertainment and viral posts. |
| **Tech Sage** | Knowledgeable, analytical, educational. Ideal for tech content and tutorials. |
| **Hype Beast** | Energetic, motivational, trend-focused. Great for marketing. |
| **Honest Critic** | Balanced, objective, thoughtful. Perfect for reviews and analysis. |
| **Quirky Observer** | Unique perspective, creative insights. Thought-provoking content. |
| **Brand Storyteller** | Narrative-focused, brand-building. Brand marketing and storytelling. |
| **Community Problem Solver** | Helpful, solution-oriented. Customer support and community engagement. |
| **Growth Strategist** | Business-focused, strategic. B2B content and growth marketing. |
| **Trend Analyst** | Forward-thinking, trend-spotting. Industry analysis and predictions. |
| **Engagement Specialist** | Relationship-building, community-focused. Engagement and relationship management. |
| **Product Evangelist** | Product-focused, feature-explaining. Product marketing and demos. |
| **Custom** | Full customization for your own unique personality. |

---

## Agent Evolution System

Agents evolve based on performance and engagement:

- **Novice (N)** – Starting level, basic capabilities
- **Advanced (A)** – Improved performance and features
- **Expert (E)** – High-level capabilities and customization
- **Legendary (L)** – Maximum performance and unique abilities

---

## Managing Your Agents

- **Agent Dashboard** – View all agents, performance metrics, and evolution status in one place.
- **Performance Tracking** – Monitor engagement rates, content performance, and evolution progress.
- **Agent Customization** – Update settings, personality traits, and content preferences.

Use [Agent Teams](/docs/guides/agent-teams) to group agents (e.g. Content, Support) and use them in [Workflows](/docs/guides/workflows).
