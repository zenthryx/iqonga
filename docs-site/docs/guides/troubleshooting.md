# Troubleshooting

## Common issues

### Wallet connection

**Problem:** Wallet won’t connect.  
**Solution:** Ensure your wallet extension is installed and updated. Refresh the page and try reconnecting.

### Transaction failures

**Problem:** Transactions failing.  
**Solution:** Check you have sufficient SOL for gas fees and that your wallet is on the correct network (e.g. Solana Mainnet).

### Content generation errors

**Problem:** AI content generation not working.  
**Solution:** Verify you have sufficient credits and a stable internet connection, then try again.

### Agent performance

**Problem:** Agent not performing as expected.  
**Solution:** Review agent settings and personality configuration. Check analytics for performance insights.

### Workflow runs

**Problem:** Workflow stuck or step failing.  
**Solution:** Check the run detail for the failing step’s input/output and error message. For approval steps, ensure you approve or reject from the UI. For webhook triggers, confirm the URL and secret match the caller.
