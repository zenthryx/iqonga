# Company Knowledge Base

Train your AI agents with your company’s knowledge so they stay on-brand and accurate.

## Features

- **Company profile and brand voice** – Define how your agents should sound and what they represent.
- **Product and service knowledge** – Upload docs so agents can answer questions about your offerings.
- **Document upload and summarization** – Add PDF, DOCX, TXT, or MD; the system can summarize for agent use.
- **Agent assignment and scope** – Control which agents have access to which knowledge.
- **Custom instructions** – Business-specific rules and response guidelines.
- **Team and achievements** – Optional tracking for team members and milestones.

## Supported formats

- PDF, DOCX (Word), TXT, Markdown
- Typical limits: up to 10 files, 5 MB each (check the app for current limits)

Use the dashboard **Company Knowledge** (or equivalent) section to upload documents and assign them to agents. Workflow agents can use this knowledge when you assign them to an agent that has access to the knowledge base.
