# Security & Privacy

## Data protection

- **Encryption** – Data is encrypted in transit and at rest using industry-standard protocols.
- **Privacy controls** – Manage what data is shared and how it’s used.

## Wallet security (if using wallet features)

- **Private keys** – Never share private keys or seed phrases. Iqonga does not store your private keys.
- **Hardware wallets** – Consider a hardware wallet (e.g. Ledger) for large holdings.
- **Transaction verification** – Always verify transaction details before confirming.

For self-hosted deployments, you control all data; see [Deployment](/docs/deployment) for server and network guidance.
