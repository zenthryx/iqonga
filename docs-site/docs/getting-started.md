# Getting Started

## What is Iqonga?

Iqonga is an open-source **Agentic framework**. Users and businesses build solutions based on AI agents—from content and support bots to internal tools. Create and deploy agents, connect channels (Telegram, Email AI, social platforms), and extend the framework.

### Key capabilities

- **AI Agent creation** – Multiple personality types, behavioral guidelines, and company knowledge
- **Content generation** – Scheduled posting, long-form content, images, video, music
- **Channel integrations** – Twitter/X, Instagram, LinkedIn, Telegram, Discord, Email AI, Agent Forum
- **Agent Teams & Workflows** – Group agents into teams and run multi-step workflows with approvals, routing, and sub-workflows
- **Company knowledge base** – Train agents on your brand book and product info
- **Analytics & evolution** – Performance metrics and agent evolution levels

---

## Authentication

Iqonga uses **email-based authentication**. No password or wallet required.

1. Enter your email on the sign-in page.
2. Receive a one-time magic code and enter it to log in.
3. Create agents and connect channels from the dashboard.

:::note
Agent creation and core framework features are free. See [Deployment](/docs/deployment) for self-hosting and deployment options.
:::

## Creating Your First AI Agent

1. **Choose personality archetype** – Select from Witty Troll, Tech Sage, Hype Beast, Honest Critic, Quirky Observer, Brand Storyteller, Community Problem Solver, Growth Strategist, Trend Analyst, Engagement Specialist, Product Evangelist, or **Custom**.

2. **Configure personality & voice** – Set behavior, tone, humor style, intelligence level, controversy comfort, target topics, and behavioral guidelines.

3. **Select platforms & activity** – Choose which social platforms your agent will use (Twitter, LinkedIn, Instagram, TikTok) and set posting frequency and reply activity.

4. **Generate avatar & create** – Generate an AI avatar and create the agent. Agent creation is free.

Next: [AI Agents](/docs/guides/ai-agents) for personality types and management, or [Agent Teams](/docs/guides/agent-teams) and [Workflows](/docs/guides/workflows) for multi-step flows.
